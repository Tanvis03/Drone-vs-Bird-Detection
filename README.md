# Drone vs Bird Detection

## Project Overview
This project uses deep learning and computer vision techniques to detect and classify drones and birds from aerial images or video streams. The system helps improve surveillance by accurately distinguishing drones from birds, reducing false detections.

## Features
- Detects drones and birds
- Image classification using deep learning
- Useful for surveillance and security applications

## Technologies Used
- Python
- TensorFlow/Keras
- OpenCV
- NumPy
- Matplotlib

## Future Enhancements
- Real-time video detection
- Improved model accuracy
- Deployment as a web application
